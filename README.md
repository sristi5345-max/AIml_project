# 🎓 Smart GPA Tracker & Grade Calculator with ML Insights

## 📌 Project Overview
The **Smart GPA Tracker & Grade Calculator** is a Python-based application designed to help students efficiently manage their academic performance. It allows users to calculate GPA, track semester-wise progress, and gain intelligent insights using Machine Learning techniques.

This project goes beyond basic GPA calculation by integrating predictive analytics and performance clustering to help students make informed academic decisions.

---

## 🚀 Features

- 📊 **GPA Calculation**
  - Cumulative GPA
  - Semester-wise GPA

- 🧾 **Course Management**
  - Add courses with credits, grades, and semester
  - Supports both letter grades and numeric scores

- 📋 **Interactive Dashboard**
  - Displays courses, GPA, and grade distribution
  - ASCII-based visual representation

- 🤖 **ML Insights & Predictions**
  - GPA trend analysis
  - Future GPA prediction using Linear Regression
  - Course performance grouping using K-Means Clustering

- 🔮 **What-if Analysis**
  - Predict GPA for future scenarios

- 📁 **Export Reports**
  - Export complete data as CSV file

## ⚙️ Installation & Setup

1. **Clone the repository**
```bash
git clone <your-repo-link>
cd smart_gpa_tracker
```
2. **Install dependencies**
   ```bash
   pip install numpy scikit-learn
   ```
3. Run the application
 ```bash
python main.py
 ```

## 📊 Machine Learning Techniques Used

### 🔹 Linear Regression
- Predicts future GPA trends based on past performance  

### 🔹 K-Means Clustering
- Groups courses into performance tiers:
  - High Performers  
  - Mid Tier  
  - Needs Improvement  

### 🔹 Label Encoding
- Converts grade categories into numerical form
